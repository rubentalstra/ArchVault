# Roadmap

This document tracks planned features, improvements, and ideas for ArchVault.
Items are grouped by priority and status. Check the boxes as work is completed.

## In Progress

Features currently being worked on or recently shipped (pending polish).

- [ ] **Autosave** — Auto-persist diagram changes *(implemented, needs UI/UX improvements)*
- [ ] **Keyboard Shortcuts** — Hotkeys for common editor actions *(implemented, needs UI/UX improvements)*

## Up Next

High-priority features planned for the near term.

- [ ] **Undo / Redo** — Full history stack for diagram edits
- [ ] **Node Focus & Flow Animation** — Click-to-focus navigation and animated data flows
- [ ] **Versions & Timeline** — Snapshot diagrams at points in time and browse history

## Planned

Features on the roadmap but not yet scheduled.

- [ ] **Tag Perspectives** — Filter diagrams by tag to create focused views
- [ ] **Dependencies View** — Auto-generated dependency graph from connections
- [ ] **Global Search** — Search across elements, connections, and diagrams
- [ ] **Import / Export** — JSON and Structurizr DSL import/export
- [ ] **Technology Catalog** — Curated library of technologies with icons and metadata
- [ ] **Activity Log** — Audit trail of changes per workspace
- [ ] **Trash & Restore** — Soft-deleted items with a recovery UI
- [ ] **Docker Deploy Guide** — Production-ready Docker Compose with TLS and backups
- [ ] **E2E Testing** — Playwright test suite for critical user journeys
- [ ] **Documentation Site** — Hosted docs for setup, usage, and API reference

## Future Ideas

Longer-term ideas that may shape the direction of the project.

- [ ] **Block Schemas** — Define reusable architecture block templates
- [ ] **Official Blocks** — Ship a library of common architecture patterns
- [ ] **Block Browser** — Discover and preview blocks inside the editor
- [ ] **Save as Block** — Turn any element subtree into a reusable block
- [ ] **Community Registry** — Publish and install blocks from the community
